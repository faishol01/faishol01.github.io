Domain Baru!!!

  • Faishol27
  • 2017-07-11
    • Comment

    Hai semua!

    Di post ini, aku mau ngumumin kalo aku udah beli (sewa atau beli sih) domain baru, yakni faishol.net. (Yeeee…)
    INGAT! Domainku faishol.net bukan yang lain.

    “Shol… kamu beli domain ya?”
    “Wuss.. pasti mahal.”
    “Weee dapet darimana domainnya?”
    “Ciee..”

    Wush wushh wush.. selow seloww..
    Kawan yang aku cintai (LOL). Sebenernya domain itu gak begitu mahal kok. Kalo dihitung-hitung gak lebih dari Rp20.000,00/bulan, PER BULAN.Jadi, kalian cuma perlu nyisihin uang saku sekitar Rp670,00 tiap harinya. Mungkin karena pembayarannya per tahun, makanya kelihatan mahal. Udah kaya sales aku…

    Perlu diketahui, sebenernya yang mahal itu bukan domainnya (tergantung domain apa yang kamu beli sih). Tapi yang mahal itu sewa hostingnya sekitar Rp50.000,00/bulan, meskipun ada yang dibawah segitu. Nah blogku ini, aku taruh di GitHub Pages yang mana Free. Tapi kelemahannya, GitHub Pages hanya bisa digunakan untuk static pages, jadi gak kaya web biasanya yang full fitur.

    “Kenapa gak pake wordpress aja, kan gratis?”
    “Atau blogger?”

    Alasan utama aku gak pake dua layanan itu karena aku gak bisa neko-neko (bahasa jawa). Jadi, susah buat edit sana-sini. Meskipun, aku dimudahkan dalam menulis post. Apalagi kalo di wordpress itu aku harus bayar lagi kalau mau pakai domain sendiri. Nah daripada aku bayar dobel, mending pake GitHub Pages aja. Selain itu, aku juga bisa belajar develop web.

    Oke, itu penjelasan singkat dariku. Selamat menjelajahi blogku!
